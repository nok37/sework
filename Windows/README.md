<a id="markdown-◆-tree" name="◆-tree"></a>
#### ◆ tree
ファイル一覧を作成できるので、エビデンス格納確認に便利
```bat
> tree /f  
C:.
│  TEST1.xlsx
│  TEST2.xlsx
│
└─sub
        TEST3.xlsx
        TEST4.xlsx
```
