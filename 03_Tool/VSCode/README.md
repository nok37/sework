## 1. キーショートカット

### 1.1. 文字操作
| 内容 | ショートカットコマンド |
| ---- | ---- |
| 矩形選択 | ctrl + shift + alt + 矢印 | 
| 大文字に変換 | Shift + Alt + u (独自定義) | 
| 小文字に変換 | Alt + u (独自定義) | 


### 1.2. ファイル操作
| 内容 | ショートカットコマンド |
| ---- | ---- |
| ファイルを名前で検索 | ctrl + p |
