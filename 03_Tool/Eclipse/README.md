## ２．ショートカット
<a id="markdown-ファイル操作系" name="ファイル操作系"></a>
### ファイル操作系

```java
// リソースを開く
ctrl + shift + r

// 呼び出し階層を開く
ctrl + alt + h

// 次のエディタへ
ctrl + F6 (ctrl + e)

// 次のビュー
ctrl + F7

// 次のパースペクティブ
ctrl + F8

// ショートカット一覧表示
ctrl + shift + l

// クイックアウトライン
ctrl + o

// クイック階層
ctrl + t
```
