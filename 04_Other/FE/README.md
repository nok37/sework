# FE（基本情報）

### バイト

|  単位  |  SI基準値  |  慣用値  |
| ---- | ---- | ---- |
| キロバイト (kB)   | 10^3  | 2^10 |
| メガバイト (MB)   | 10^6  | 2^20 |
| ギガバイト (GB)   | 10^9  | 2^30 |
| テラバイト (TB)   | 10^12 | 2^40 |
| ペタバイト (PB)   | 10^15 | 2^50 |
| エクサバイト (EB) | 10^18 | 2^60 |
| ゼタバイト (ZB)   | 10^21 | 2^70 |

* 慣用値とは  
10^21 = 1000^7 = 1,000,000,000,000,000,000,000  
2^70 = 1024^7 = 1,180,591,620,717,411,303,424

* ZFS(Zettabyte File System)  
Oracle開発の大規模ファイルシステム
